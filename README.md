# 🛰️ SkillGap Radar — AI-Driven Career Laboratory

**Know what’s missing before you apply. Bridge the gap with scientific precision.**

SkillGap Radar is a production-grade career optimization platform designed to eliminate the guesswork from job applications. Leveraging advanced AI-driven analysis, cinematic 3D storytelling, and dynamic roadmap generation, it transforms your professional trajectory from static to strategic.

---

## 🦄 Laboratory Features

### 1. Cinematic 3D Entry Portal
- **High-Fidelity Interaction**: Experience a mandatory "Experience-First" gateway powered by **UnicornStudio 3D**.
- **Reload-Proof Gating**: Every fresh session is initialized through a secure 3D lifecycle to ensure a premium laboratory entry.

### 2. Intelligent PDF Vector Mapping
- **Stable Extraction Engine**: Robust client-side PDF parsing using the **Vite-Native Worker Pattern** (`pdfjs-dist`).
- **Precision Analysis**: Compares your core competencies against a specific Job Description to find the exact "Gap Delta."

### 3. Dynamic Optimization Roadmap
- **Phase-Based Strategy**: Generates a 3-phase action plan (Critical ATS, Content Resonance, Competitive Trajectory).
- **Actionable Operational Commands**: Provides the logic, the command, and the estimated match probability boost for every identified gap.

### 4. Zero-G Transitions
- **Global Shutter System**: Seamless, laboratory-grade page transitions with scanning animations and system status readouts.

---

## 🛠️ Core Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite 8](https://vitejs.dev/)
- **Visuals**: [Framer Motion](https://www.framer.com/motion/) + [UnicornStudio](https://unicornstudio.io/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) + Custom Glassmorphism System
- **PDF Engine**: [PDF.js](https://mozilla.github.io/pdf.js/) (Bundled Local Worker)
- **Icons**: [Lucide React](https://lucide.dev/)

## ⚡ Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/Helix-1716/SkillGap-Radar.git
cd SkillGap-Radar
npm install
```

### 2. Enter Development Mode
```bash
npm run dev
```

### 3. Build for Release
```bash
npm run build
```
